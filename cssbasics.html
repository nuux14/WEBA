CSS Basics

CSS stands for Cascading Style Sheets. It's a language used to control the visual appearance of web documents.

The three different ways to add CSS to an HTML file

  1. Inline HTML style attribute
  2. Internal HTML style tag
  3. External CSS file

Sources of CSS styles

  1. Inline style
  2. Internal style
  3. External stylesheet
  4. User agent (Browser defaults) stylesheet

CSS Rules

CSS is composed of rules. A rule is composed of selector and declaration. Selector tells the browser which HTML element to apply the style to. Declaration on the other side is a set of pairs of css properties and values.
